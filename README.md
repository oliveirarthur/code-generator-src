# Code Generator

## Stack

- Frontend
	- web
		- Angular
		- TypeScript
		- Yarn

## Instalation

- clone the project
	- `git clone git@github.com:oliveirarthur/code-generator.git`
- install dependencies
	- `yarn install`
- get the containers up and running
	- `docker-compose up --build`

## Default development addresses

- Web frontend
	- http://localhost:4200
